# Grammar

这里不会把语法当作考试规则，而是解释英语为什么需要这些结构，以及母语者什么时候会使用它们。

本模块将覆盖：

- Passive Voice
- Modal Verbs
- Articles
- Prepositions
- Gerunds and Infinitives
- Relative Clauses
- Conditionals

:::tip One memorable rule
语法的作用不是限制表达，而是帮助听者更准确地理解你的意思。
:::

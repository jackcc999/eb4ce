# Tenses

时态不是一张需要死记的变形表，而是说话者看待时间和状态的方式。

本模块会重点解释：

- 为什么英语必须明确时间关系
- 一般时、进行时和完成时分别强调什么
- Present Perfect 为什么经常让中文母语者困惑
- 工作沟通中如何选择自然的时态

## Recommended order

1. Present Simple
2. Present Continuous
3. Present Perfect
4. Past Simple
5. Past Continuous
6. Past Perfect
7. Future Forms

:::tip One memorable rule
先判断你想强调的是事实、过程，还是与现在有关的结果。
:::

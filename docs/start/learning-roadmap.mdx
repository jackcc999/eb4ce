# Learning Roadmap

EB4CE 的学习顺序不是按照传统语法书排列，而是按照理解依赖关系组织。

## Stage 1 — Sentence Structure

先理解英语句子的骨架：

- Sentence
- Subject
- Verb
- Object
- Complement
- Modifier
- Phrase
- Clause
- Word Order

完成这一阶段后，你应该能够快速回答：

> 这个句子的核心动作是什么？谁做了这件事？其他部分在补充什么？

## Stage 2 — Sentence Patterns

学习英语中重复出现的高频表达结构，例如：

- `be`
- `have`
- `There be`
- `have to`
- `need to`
- `used to`

目标不是背句型名称，而是看到场景时自然选择结构。

## Stage 3 — Tenses

建立“动作与时间关系”的 mental model：

- Present
- Past
- Future
- Continuous
- Perfect

时态不是时间标签，而是说话者选择观察事件的方式。

## Stage 4 — Grammar

在句子骨架和时间视角稳定后，再学习：

- Voice
- Articles
- Prepositions
- Questions
- Modals
- Conditionals

这时语法规则会变成对既有直觉的总结，而不是孤立规则。

## Stage 5 — Think in English

识别中文直译带来的常见问题，并练习直接表达意思：

- 信息重点
- 动作视角
- 主语选择
- 自然语序
- 简洁表达

## Stage 6 — Engineer English

把前面的能力应用到真实工作中：

- Slack
- Email
- Standup
- Pull Request
- Code Review
- Design Review
- Incident
- Interview
- AI Discussion

## 推荐方式

不要一次阅读大量文章。每次完成一个概念，并确保你可以：

1. 用自己的话解释 mental model；
2. 识别三个真实例子；
3. 修正一个常见错误；
4. 在工作中主动使用一次。

# How to Use EB4CE

EB4CE 不是一本需要从头背到尾的语法书。它是一张互相连接的知识地图。

## 先理解，再记结构

阅读一篇文章时，按这个顺序进行：

1. **Question**：这篇文章解决什么问题？
2. **Why**：为什么英语需要这样表达？
3. **Mental Model**：应该如何在脑中理解？
4. **Pattern**：真实表达通常长什么样？
5. **Examples**：它在工程工作中如何出现？
6. **Mistakes**：中文母语者为什么容易说错？
7. **Practice**：你能否自己生成新句子？

不要只记最后的公式。如果 mental model 不清楚，公式很快会被忘记。

## 一次只学一个概念

每篇核心文章只围绕一个主要概念。完成后再进入 Next Step。

判断自己是否真正理解，可以尝试：

- 不看文章，用中文解释这个概念；
- 写出三个不同场景的英文例子；
- 解释一个错误句子为什么不自然；
- 在 Slack、会议或邮件里主动使用。

## 使用知识连接

文章底部会逐步提供三类连接：

- **Prerequisites**：理解本文前最好先掌握什么；
- **Related**：哪些概念可以帮助你比较和扩展；
- **Next Step**：接下来最自然的学习节点。

遇到困难时，不要反复硬读当前页面。先返回 prerequisite，补齐缺失概念。

## 把输入变成输出

只阅读不会自动形成表达能力。每学完一个概念，至少完成一次输出：

```text
今天学到的核心直觉：

我以前容易说错的是：

我现在会这样表达：

我准备在哪个工作场景使用：
```

## 不需要追求一次掌握

英语直觉来自多次接触同一模式。第一次目标是理解，第二次目标是识别，第三次目标才是自然使用。

# Sentence Patterns

真正开口表达时，人们通常不是先分析语法，而是直接调用熟悉的句型。

本模块会讲清楚 `be`、`have`、`there be`、`have to` 等高频结构的含义、使用场景和常见错误。

## Recommended order

1. be
2. have
3. There be
4. be going to
5. have to
6. need to
7. used to
8. would like to

:::tip One memorable rule
把句型当作可以直接复用的代码模板，而不是零散的语法规则。
:::

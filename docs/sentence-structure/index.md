# English Sentence Structure

英语表达的第一步，不是背单词，而是先看清一句话的骨架。

本模块会帮助你理解：

- 主语、动词、宾语和补语分别做什么
- 为什么英语通常不能像中文一样省略主语
- 为什么英文语序比中文更固定
- 如何快速定位一句话的核心结构

## Recommended order

1. What Is a Sentence?
2. Subject
3. Verb
4. Object
5. Complement
6. Modifier
7. Word Order

:::tip One memorable rule
先找动词，再找是谁做这件事。
:::

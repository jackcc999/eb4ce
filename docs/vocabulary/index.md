# Vocabulary

词汇不应该孤立记忆，而应该和场景、搭配与语气一起学习。

本模块会整理工程师高频词汇、常见搭配、近义表达和容易误用的单词。

重点包括：

- Engineering vocabulary
- Meetings and collaboration
- Project updates
- AI and ranking terminology
- Common collocations
- Natural alternatives

:::tip One memorable rule
不要只记一个单词的中文意思，要同时记住它通常和哪些词一起出现。
:::

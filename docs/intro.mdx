# Build English Intuition

**English Bible for Chinese Engineers** 是一个为中文母语工程师打造的开源英语学习系统。

这里不会从一堆语法术语开始。我们先解释英语为什么这样表达，再总结规则，并把它放进 Slack、会议、代码评审、邮件和技术面试等真实场景中。

## 你会在这里学到什么

- 看见句子时，快速识别它的骨架；
- 理解时态背后的时间视角，而不是只背公式；
- 减少逐字翻译，直接组织英语表达；
- 在工程工作中说得自然、清晰、准确。

## 推荐学习路径

1. [查看 Learning Roadmap](./start/learning-roadmap.mdx)，了解完整学习顺序。
2. 从 **Sentence Structure** 开始，建立句子骨架。
3. 学习高频 **Sentence Patterns** 和 **Tenses**。
4. 进入 **Think in English**，修正中文思维造成的表达偏差。
5. 在 **Engineer English** 中应用到真实工作场景。

## 每篇核心文章的结构

一篇文章只解决一个核心问题，并尽量包含：

- **Why**：为什么英语需要这样表达；
- **Mental Model**：应该如何在脑中理解；
- **Pattern**：实际使用结构；
- **Engineer Examples**：工程师工作中的真实例子；
- **Common Mistakes**：中文母语者常见错误；
- **Next Step**：下一步应该学习什么。

> Don’t translate word by word. Understand the meaning, then choose the expression.

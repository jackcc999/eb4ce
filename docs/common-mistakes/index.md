# Common Mistakes

很多错误不是因为你“不懂英语”，而是因为中文和英语在结构、时间和表达习惯上不同。

本模块会重点拆解：

- be done vs. have done
- There be vs. have
- say vs. tell
- in / on / at
- do vs. make
- much vs. many
- since vs. for

每篇文章都会解释错误为什么发生，以及母语者会如何思考。

:::tip One memorable rule
只记正确答案不够，还要知道错误答案为什么听起来不自然。
:::

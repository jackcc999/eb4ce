# Think in English

中文和英语组织信息的方式不同。很多“不自然”的表达，并不是单词或语法错了，而是把中文思维直接翻译成了英文。

本模块会帮助你理解：

- 为什么英语更强调明确的主语
- 为什么逐字翻译经常不自然
- 母语者如何根据语境选择表达
- 如何从“翻译中文”转向“表达意思”

:::tip One memorable rule
不要先想中文句子，再逐字替换；先确认你真正想传递的信息。
:::

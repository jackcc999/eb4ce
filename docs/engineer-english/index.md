# Engineer English

这里收集软件工程师在真实工作中最常用的英语表达。

你会学到如何在以下场景中自然沟通：

- Daily Standup
- Slack
- Email
- Code Review
- Design Review
- Debugging
- AI and Ranking Discussions
- Interviews
- Performance Reviews

:::tip One memorable rule
工作英语最重要的不是复杂，而是清楚、准确、容易采取行动。
:::
